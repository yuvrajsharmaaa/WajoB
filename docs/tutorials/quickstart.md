## ğŸ†˜ Quick Troubleshooting

### Wallet won't connect?
1. Update Telegram to latest version
2. Update wallet app to latest version
3. Try different wallet (Tonkeeper â†’ Tonhub)
4. Restart both apps

### Transaction stuck?
1. Wait 2 minutes (blockchain confirmation time)
2. Check [tonscan.org](https://tonscan.org) with transaction hash
3. If >5 minutes, retry transaction

### Job not appearing?
1. Pull down to refresh
2. Check filters (may be hiding it)
3. Ensure escrow is funded
4. Wait 30 seconds for sync

### Can't apply to job?
1. Check if already applied
2. Verify job still open (not filled)
3. Ensure wallet connected
4. Check you're not the employer
