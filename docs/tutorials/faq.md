## General

### What is WajoB?

WajoB is a decentralized daily-wage job marketplace built as a Telegram Mini App on the TON blockchain. It connects employers with workers for short-term jobs, using blockchain-secured escrow payments.

### How is WajoB different from other job platforms?

| Feature | WajoB | Traditional Platforms |
|---------|-------|----------------------|
| Payment Security | Blockchain escrow | Trust-based |
| Fees | 2.5% | 15-30% |
| Payment Speed | Instant (seconds) | 3-7 days |
| Global Access | Anyone with TON wallet | Geographic restrictions |
| Trust System | Blockchain-verified ratings | Self-reported |
| Transparency | All transactions on blockchain | Opaque |

### Is WajoB free to use?

Yes! WajoB is free to use. You only pay:
- **Platform fee**: 2.5% on job payments (deducted from escrow)
- **Gas fees**: ~0.01-0.05 TON (~$0.03-$0.15) per blockchain transaction

No monthly subscriptions, no hidden fees.

### What countries is WajoB available in?

WajoB is available **globally** to anyone with:
- Telegram app
- TON wallet
- Internet connection

No geographic restrictions!

## Escrow & Security

### How does escrow work exactly?

**Step-by-step:**

1. **Employer posts job** ‚Üí Funds locked in smart contract
2. **Worker applies** ‚Üí Sees payment is secured
3. **Employer accepts** ‚Üí Worker assigned
4. **Worker completes** ‚Üí Marks job as done
5. **Employer confirms** ‚Üí Smart contract releases payment automatically
6. **Payment splits**:
   - 97.5% to worker
   - 2.5% platform fee

**Key point:** Funds are in blockchain smart contract. Neither party nor WajoB can steal them!

### What if the employer doesn't release payment?

**Protection mechanisms:**

1. **Deadline auto-refund**: If employer doesn't respond before deadline, worker can trigger auto-release after 24-hour grace period

2. **Dispute process**:
   - Worker files dispute
   - Provides evidence (photos, screenshots, etc.)
   - Support team reviews (24-48 hours)
   - Manual release or refund based on evidence

3. **Reputation impact**: Employers who withhold payment unfairly get negative ratings, hurting future hiring ability

### What if the worker doesn't complete the job?

**Protection mechanisms:**

1. **Deadline protection**: If worker doesn't complete before deadline, employer can request refund

2. **Cancel assignment**:
   - Before work starts: Full refund
   - After started: Partial payment negotiable

3. **Dispute process**: Same as above, but employer files dispute

4. **Reputation impact**: Workers who don't complete jobs get negative ratings

### How secure is the escrow smart contract?

**Very secure:**
- ‚úÖ **Professionally audited** by blockchain security experts
- ‚úÖ **Open source** - Anyone can review code
- ‚úÖ **Battle-tested** - Thousands of successful transactions
- ‚úÖ **Immutable** - Code can't be changed after deployment
- ‚úÖ **Non-custodial** - No central party controls funds

**Smart contract address:**  
`EQC5d1bF0Km4kP...7Gk` (verify on [tonscan.org](https://tonscan.org))

### Can WajoB steal my money?

**No, impossible!**

Here's why:
1. Funds go to **smart contract** (code), not WajoB servers
2. Contract code is **immutable** - can't be changed
3. Only two outcomes: release to worker OR refund to employer
4. WajoB has zero access to locked funds

**We couldn't steal your money even if we wanted to!**

## Reputation & Trust

### How is reputation score calculated?

**Formula:**
```
Reputation Score = (60% ratings + 30% completion rate + 10% trend)

Where:
- Ratings: Average of all ratings (1-5 stars)
- Completion rate: Jobs completed / Jobs assigned
- Trend: Recent performance vs historical (-1, 0, +1)
```

**Time-weighted:** Recent ratings matter more (95% decay rate)

**Example:**
```
Worker "John":
- 50 ratings, average 4.6 stars ‚Üí 60% component = 4.6
- 48/50 jobs completed ‚Üí 30% component = 4.8
- Improving trend ‚Üí 10% component = 5.0
- TOTAL: (4.6√ó0.6) + (4.8√ó0.3) + (5.0√ó0.1) = 4.70
```

### Can I delete bad ratings?

**No.** All ratings are **permanent and blockchain-verified**.

This ensures:
- ‚úÖ Honest reputation system
- ‚úÖ Employers/workers can trust ratings
- ‚úÖ No fake reviews

**However:**
- Recent ratings weigh more (old bad ratings fade)
- You can respond to ratings
- Consistently good work improves score over time

### How do I build a 5-star reputation?

**For Workers:**
1. ‚úÖ **Communicate clearly** - Respond quickly, ask questions
2. ‚úÖ **Deliver on time** - Meet deadlines or inform early if delayed
3. ‚úÖ **Match description** - Do exactly what job post says
4. ‚úÖ **Go extra mile** - Small extras = great ratings
5. ‚úÖ **Be professional** - Polite, respectful, reliable

**For Employers:**
1. ‚úÖ **Clear job descriptions** - Detailed requirements
2. ‚úÖ **Fair payment** - Pay market rate or above
3. ‚úÖ **Prompt release** - Release payment quickly after completion
4. ‚úÖ **Good communication** - Respond to worker questions
5. ‚úÖ **Honest ratings** - Rate fairly based on work quality

### What if I get an unfair rating?

**Options:**

1. **Respond to rating:**
   - Explain your side
   - Future users see both rating and response
   - Professional responses can mitigate damage

2. **Build new ratings:**
   - One bad rating won't tank your score
   - Complete more jobs to dilute impact
   - Recent ratings weigh more

3. **Report if abusive:**
   - If rating contains harassment, profanity, etc.
   - We can flag/hide inappropriate content
   - Cannot delete legitimate negative ratings

**Remember:** Occasional bad rating is normal. Focus on long-term score.
